Meilleur k pour l'algorithme k-NN

Ce programme sert à déterminer quel paramètre K pour l'algorithme k-NN est le plus approprié pour un certain dataset. Vous aurez besoin d'un set contenant les données et d'un set d'entraînement qui prend certains éléments du dataset afin de voir si l'algorithme est capable de trouver la bonne catégorie.
